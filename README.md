# 3D Exploded View Project

This project is a 3D visualization of an exploded view using Three.js.

## Setup

1.  **Install Dependencies**:
    The project is already set up. If you need to reinstall:
    ```bash
    npm install
    ```

2.  **Run Development Server**:
    Start the local development server to view the project:
    ```bash
    npm run dev
    ```

3.  **Build for Production**:
    ```bash
    npm run build
    ```

## Features

-   **Three.js**: Core 3D engine.
-   **GSAP**: Smooth animation for the explosion effect.
-   **Lil-GUI**: Debug panel to control explosion parameters manualy.
-   **OrbitControls**: Rotate, zoom, and pan around the model.

## Structure

-   `src/main.js`: Main application logic (scene setup, exploded view logic).
-   `src/style.css`: Basic styling.
-   `index.html`: Entry point.
